பூஜா விதானம்

(எந்த பூஜை யாக இருந்தாலும் சிறிய பிள்ளையார் தயார் செய்துகொண்டு - பூஜையையும் செய்ய வேண்டும்)
 சிறிய பிள்ளையார்
  • 2 வெற்றிலை - அதற்கு மேல் - நெல்லிக்காய் அளவு மஞ்சளால் ஆன பிள்ளையார் ஐ செய்து வைக்கவும்
  • பத்தி மாலை அணியவும்
  • 2 பழங்கள் வைக்கும் (சிறிய பிள்ளையார்)

    முதலில் சிறிய பிள்ளையாருக்கு பூஜை (First small pillayaar pooja)
     சுக்லாம்பரதரம் விஷ்ணும்
    சசிவர்ணம் சதுர்புஜம்
    பிரஸந்ந வதனம் த்யாயேத்
    ஸர்வ விக்னோப சாந்தயே
    ஓம்
    பூர்: புவ: ஸுவ:
    தத் ஸவிதுர் வரேண்யம்
    பர்கோ தேவஸ்ய தீமஹி
    தியோ: யோந: ப்ரசோதயாத்

    ததேவ லக்னம் சுதினம் ததேவ
    தாராபலம் சந்த்ர பலம் ததேவ
    வித்யாபலம் தைவபலம் ததேவ
    லக்‌ஷ்மீ பதேங்க்ரீ யுகம் ஸ்மரராமி

    ஆசன்ம்யாம்
    உத்தரணியில் தீர்த்தம் எடுத்து, வலது உள்ளங்கையில் விட்டுக் கொண்டு, இதை சொல்லி என்று சொல்லி, மூன்றுமுறை உட்கொள்ள வேண்டும்
     ஓம் கேசவாய ஸ்வாஹா
    ஓம் நாராயணாய ஸ்வாஹா
    ஓம் மாதவாய ஸ்வாஹா.

    Dhyaanam (keep praying offering flower to God)
     1. த்யாயாமி ( பூ அல்லது அட்ஷதை )
    2. ஆவாஹயாமி ( பூ அல்லது அட்ஷதை )
    3. ஆஸனம் ஸமர்ப்பயாமி ( பூ அல்லது அட்க்ஷதை )
    4. ஹஸ்தயோஹோ அர்க்யம் ஸமர்பயாமி ( தீர்த்தம் )
    5. பாதயோஹோ பாத்யம் ஸமர்பயாமி ( தீர்த்தம் )
    6. மதுவர்கம் ஸமர்பயாமி ( பால் )
    7. பஞ்சாம்ருத ஸ்நாநம் ஸமர்பயாமி ( பஞ்சாம்ருதம் )
    8. ஸூத்தோதக ஸ்நாநம் ஸமர்பயாமி ( தீர்த்தம் )
    9. ஸ்நானான்தரம் ஆசமனியம் ஸமர்பயாமி ( தீர்த்தம் )
    10. யக்யோப வீதார்த்தம் அட்க்ஷதாம் ஸமர்பயாமி ( அட்சதை )
    11. திவ்ய பரிமள கந்தாம் தாரயாமி ( பூவில் சந்தனம் தோய்த்து சமர்ப்பிக்கவும் . )
    12. கந்தஸ்யோபரி குங்குமம் ஸமர்பயாமி ( பூவில் குங்குமம் தோய்த்து சமர்பிக்கவும் )
    13. அலங்கரணார்த்தம் அட்க்ஷதாம் ஸமர்பயாமி ( அட்சதை )
    14. மால்யார்த்தம் புஷ்பைஹி பூஜயாமி ( புஷ்பம் )
