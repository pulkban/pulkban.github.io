Reverse the string: 


public class Main {
    public static void main(String[] args) {
        String str = "hello world";
        char[] chars = str.toCharArray();
        int n = chars.length;

        // Reverse in place
        for (int i = 0; i < n / 2; i++) {
            char temp = chars[i];
            chars[i] = chars[n - 1 - i];
            chars[n - 1 - i] = temp;
        }

        String reversed = new String(chars);
        System.out.println("Original: " + str);
        System.out.println("Reversed: " + reversed);
    }
}

Bring 3rd largest num in array - using streams 


import java.util.Arrays;
import java.util.Comparator;

public class Main {
    public static void main(String[] args) {
        int[] arr = {5, 1, 9, 3, 7, 9};

        int someNumber = Arrays.stream(arr)
                               .boxed()
                               .sorted(Comparator.reverseOrder())
                               .distinct()
                               .skip(2)
                               .findFirst()
                               .orElseThrow(() -> new RuntimeException("message"));

        System.out.println("someNum: " + someNumber);
    }
}

In main(), create two threads to safely increment a counter and also update a ConcurrentHashMap atomically. Print the counter and map. 


import java.util.concurrent.ConcurrentHashMap;

public class Main {
    public static void main(String[] args) throws InterruptedException {
        final Object lock = new Object();
        int[] counter = {0}; // use array for effectively final in lambda

        Runnable task = () -> {
            synchronized (lock) { counter[0]++; }
        };

        Thread t1 = new Thread(task);
        Thread t2 = new Thread(task);
        t1.start(); t2.start();
        t1.join(); t2.join();

        System.out.println("Counter: " + counter[0]);

        ConcurrentHashMap<String, Integer> map = new ConcurrentHashMap<>();
        map.put("A", 1);
        map.merge("A", 1, Integer::sum); // atomic update
        System.out.println("Map: " + map);
    }
}

Write an EventListener Code: Lets a method listen to events published in your Spring application. - Decouples the publisher (sends event) and listener (handles event) - Can be synchronous or asynchronous (@Async). 


// Main class
public class MainApp {

    public static void main(String[] args) {
        System.out.println("Publishing event...");

        // Create event
        MyEvent event = new MyEvent("Hello Event!");

        // Notify listener
        MyEventListener listener = new MyEventListener();
        listener.handleEvent(event);
    }
}

// Custom event
class MyEvent {
    private final String message;

    public MyEvent(String message) {
        this.message = message;
    }

    public String getMessage() {
        return message;
    }
}

// Event listener
class MyEventListener {
    public void handleEvent(MyEvent event) {
        System.out.println("Received event with message: " + event.getMessage());
    }
}

Distributed Locking in Spring Boot 


Used when multiple instances of a service must not execute the same critical section at the same time.
Examples: scheduled jobs, updating shared resource, payment processing.
Common implementations: Redis, Zookeeper, Database locks.

import org.redisson.api.RLock;
import org.redisson.api.RedissonClient;
import org.springframework.stereotype.Service;

import java.util.concurrent.TimeUnit;

@Service
public class JobService {

    private final RedissonClient redisson;

    public JobService(RedissonClient redisson) {
        this.redisson = redisson;
    }

    public void runJob() {
        RLock lock = redisson.getLock("myJobLock");
        try {
            // try lock for 5 sec, auto release after 10 sec
            if (lock.tryLock(5, 10, TimeUnit.SECONDS)) {
                System.out.println("Running critical job");
                Thread.sleep(2000); // simulate work
            } else {
                System.out.println("Job already running elsewhere");
            }
        } catch (InterruptedException e) {
            e.printStackTrace();
        } finally {
            if (lock.isHeldByCurrentThread()) lock.unlock();
        }
    }
}

Microservices ACL / Anti-Corruption Layer (ACL) pattern — simple hint style: 


Purpose: Protect your service from legacy or external system models; prevent “leaking” unwanted complexity into your domain.
How: Introduce a translator layer that converts external models → internal models.
Benefit: Keeps your microservice clean, consistent, and decoupled.

// External service DTO
class ExternalUser {
    public String fullName;
    public String phone;
}

// Internal domain model
class User {
    private String name;
    private String contactNumber;
}

// ACL / Translator
class UserTranslator {
    public static User fromExternal(ExternalUser e) {
        User u = new User();
        u.setName(e.fullName);
        u.setContactNumber(e.phone);
        return u;
    }
}

// Usage
ExternalUser ext = new ExternalUser();
ext.fullName = "Alice"; ext.phone = "12345";

User internal = UserTranslator.fromExternal(ext);

=================SPECIFICATION (Dynamic Filter)
import org.springframework.data.jpa.domain.Specification;

public class EmployeeSpecification {

    public static Specification filter(
            String name, String dept, Double minSalary, Double maxSalary) {

        return (root, query, cb) -> {
            var predicates = cb.conjunction();

            if (name != null && !name.isEmpty()) {
                predicates = cb.and(predicates,
                        cb.like(root.get("name"), "%" + name + "%"));
            }

            if (dept != null && !dept.isEmpty()) {
                predicates = cb.and(predicates,
                        cb.equal(root.get("department"), dept));
            }

            if (minSalary != null) {
                predicates = cb.and(predicates,
                        cb.greaterThanOrEqualTo(root.get("salary"), minSalary));
            }

            if (maxSalary != null) {
                predicates = cb.and(predicates,
                        cb.lessThanOrEqualTo(root.get("salary"), maxSalary));
            }

            return predicates;
        };
    }
}

import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;
import java.util.List;

@Service
public class EmployeeService {

    @Autowired
    private EmployeeRepository repo;

    public List getEmployees(
            String name, String dept, Double minSalary, Double maxSalary) {

        return repo.findAll(
            EmployeeSpecification.filter(name, dept, minSalary, maxSalary)
        );
    }
}
