Chain of Decorators?
using chain of decor, write code for login start, user auth, money transf, log ended?

def logging(func):
    def wrapper():
        print("logging started")
        func()
        print("logging ended")
    return wrapper

def authentication(func):
    def wrapper():
        print("user authenticated")
        func()
    return wrapper

@logging
@authentication
def transfer_money():
    print("money transferred")

transfer_money()

output---
logging started
user authenticated
money transferred
logging ended

--meaning: transfer_money = logging(authentication(transfer_money))

        

Method Overloading = Compile-time Polymorphism
Method Overriding = Runtime Polymorphism

Override e.g.:
class Animal:
    def sound(self):
        print("animal sound")

class Dog(Animal):
    def sound(self):
        print("bark")

a = Dog()
a.sound()
output---- bark
[same method -> different forms]

Overloading e.g.:
class Calc:
    def add(self, a, b, c=0):
        return a + b + c
x = Calc()
print(x.add(1, 2))
print(x.add(1, 2, 3))
output---- 3   6
[same method -> different behavior based on arguments]
