    • Exams

  • KCET → Eligibility 45% PCM → Rank = 50% KCET + 50% Board PCM → Exam: 60+60+60 (P/C/M) → No negative marking → RVCE CSE cutoff ~Rank <500; PES ~<800; BMS ~<1200; MSRIT ~<1500 → Fees ~₹90K–1.5L/year →
  • COMEDK → Eligibility 45% PCM → Rank = Entrance only → Exam: 60+60+60 (P/C/M) → No negative marking → Seats ~30–35% in RVCE/PES/BMS/MSRIT → Fees ~₹2–3.5L/year.
  • VITEEE → Eligibility 60% PCM → Exam: 35+35+40+10+5 = 125 Qs (Phy/Chem/Math/Apt/Eng) → +1 / -1 → Fees ~₹1.8–2.5L/year.
  • BITSAT → Eligibility 75% PCM + 60% each → Exam: 45+45+45+10+10 = 155 Qs (Phy/Chem/Math/Eng/Reasoning) → +3 / -1 → BITS campuses: Pilani, Goa, Hyderabad → Fees ~₹3–3.5L/year.
  • JEE Main → Eligibility 75% aggregate OR top 20 percentile → Exam: PCM (MCQs + numerical) → +4 / -1 → NIT/IIIT → Fees ~₹1–2L/year.
  • NIT Surathkal via JEE Main (Home State quota 50% for Karnataka).
  • JEE Adv → Same eligibility → IIT → +4 / -2 → Fees ~₹2–3L/year.
  • Seats split (RVCE/PES/BMS/MSRIT) → KCET ~45–50%, COMEDK ~30–35%, Mgmt ~20–25% → No JEE.
  • Board marks role → KCET (yes, 50%), others (only eligibility).
  • Duration → All B.Tech = 4 years.

    • Law / CA

    Law:

  • 12th → CLAT UG → 5-year integrated law course → Bar Council → AIBE → Practice.
  • Already a graduate → 3-year LL.B → Bar Council → AIBE → Practice.
    CA:
  • Graduate → Register with ICAI - Direct Entry → CA Intermediate (target 60 in all 3 subjects in 2 groups = total 6) → Articleship (3 yr under CA) → CA Final → ICAI Membership → Practice.
  • 12th → (along with BCom) CA Foundation → CA Intermediate → Articleship → CA Final → ICAI Membership → Practice.
